# AiGameAgent Design

Design documentation site for **[AiGameAgent (aiGameGongfang) Studio](https://github.com/YeLuo45/AiGameAgent)** — a multi-platform mini-game development workflow monorepo that orchestrates 30+ role-based AI agents across 7 departments to deliver HTML5 page games and WeChat / Douyin mini-games.

## What's inside

- 14 documentation pages covering architecture, modules, workflows, and operations
- VitePress 1.6.4 (Markdown → static site, deployed via GitHub Pages)
- API reference, deployment guide, and module deep-dives

## Quick start

```bash
npm install
npm run docs:dev      # local dev server
npm run docs:build    # production build
```
